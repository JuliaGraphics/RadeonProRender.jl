# RadeonProRender

Julia wrapper for AMD's [RadeonProRender](https://radeon-pro.github.io/RadeonProRenderDocs/en/index.html) library.

### Installation
(Windows and Linux only right now!)
In the Julia REPL, execute

```Julia
Pkg.add("RadeonProRender")
```
